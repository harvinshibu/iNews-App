# iNews - iOS News App

iOS News Application.

## What is this?

iNews is an iOS news application which users can used to read news. NewsAPI is used in this app.

## Features

- View news in list.
- View news in details.

## Development process

- Developed using Xcode Storyboard.
- Used MVVM.
